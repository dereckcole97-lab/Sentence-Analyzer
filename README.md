# Sentence-Analyzer
## Description
Building a sentence analyzer that will essentially take a sentence and get the count for the number of words, vowels, consonants and punctuation marks
## Features
- add a sentence and you will get the count of either words, vowels, consonants or punctuation marks or all of them
- deep search for each and all the four categories
- pretty straight-forward interface
## Technologies Used
- JavaScript
